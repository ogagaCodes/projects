## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/ogagaCodes/projects.git
    ```
2. Navigate to the project directory:
    ```sh
    cd 
    ```
3. Install the dependencies:
    ```sh
    npm install
    ```
